현장 모의고사 수험표 인쇄 시스템 - Walkthrough
구현 결과 요약
Google Apps Script 기반으로 수험생이 직접 로그인하여 수험표를 인쇄할 수 있는 웹 앱을 구현했습니다. 소방 수험표 실제 양식(소방 수험표.jpg)을 분석하여 HTML 인쇄 템플릿에 반영했습니다.

파일 구조
파일	역할
Code.gs
백엔드 (라우팅, CRUD, 인증, 통계)
Index.html
수험생 로그인 + 수험표 인쇄
Admin.html
관리자 대시보드 + 데이터 관리
Css.html
글래스모피즘 UI + 인쇄 CSS
소방 수험표 양식 분석
아래 실제 양식 이미지를 기반으로 인쇄 템플릿을 구현했습니다:

소방 수험표 원본 양식

양식에 포함된 필드
성명, 생년월일, 응시번호, 응시지역, 응시분야(공채/일반경채/구급경채), 응시계급, 시험장소, 시험과목 + 안내사항

배포 방법
Google Apps Script에서 새 프로젝트 생성
Google Sheets를 만들고 Apps Script 프로젝트에 연결
각 파일을 Apps Script 에디터에 복사:
Code.gs
 → 기본 스크립트 파일
Index.html
, 
Admin.html
, 
Css.html
 → 파일 > HTML 파일 추가
배포 > 새 배포 > 웹 앱 > 배포
사용 흐름
관리자
배포URL?page=admin 접속 → 비밀번호 입력 (기본: admin1234)
수험생 정보 개별 또는 일괄 등록 (탭 복사-붙여넣기)
중요: 각 수험생의 직렬(경찰/소방)을 정확히 입력해야 합니다.
목록에서 검색/필터/수정/삭제
수험생
배포 URL 접속 → 직렬(경찰/소방) 선택 후 이름 + 연락처 입력 (3중 매칭)
본인 수험표 정보 확인
🖨️ 인쇄 버튼 클릭 → A4 규격에 최적화된 양식으로 인쇄
남은 작업
 경찰 수험표 양식 이미지 제공 후 해당 양식에 맞게 템플릿 업데이트
