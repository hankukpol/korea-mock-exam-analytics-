# UI/UX 전면 디자인 개편 계획

사용자의 피드백을 반영하여 현재의 다크 테마 가스모피즘 디자인을 폐기하고, 업계 표준의 깔끔한 비즈니스 스타일 라이트 모드로 개편합니다.

## 주요 변경 사항

### 1. 색상 체계 (Color Palette)
- **Background**: `#f8f9fa` (연한 회색 배경)
- **Main Color**: `#007bff` (진한 파란색 - 브랜드 컬러)
- **Secondary Color**: `#e9ecef` (입력창 및 카드배경)
- **Text Color**: `#212529` (가독성 높은 검은색 계열)
- **Point Color**: `#ebf1ff` (연한 파란색 - 테이블 헤더 등)

### 2. UI 구조 개편
- **카드 시스템**: 입체감 있는 쉐도우(`box-shadow`)를 활용한 화이트 카드 레이아웃 적용
- **테이블**: 선명한 회색 테두리와 연한 파란색 헤더를 가진 정갈한 테이블 구조
- **피라미드**: 화이트/블루 톤의 깔끔한 레이어 형태로 재설계
- **로그인 화면**: 깔끔한 중앙 집중형 화이트 폼 구성

### 3. 세부 컴포넌트 개선
- **차트**: 레이더 차트의 색상을 새로운 블루 테마에 맞게 조정
- **상태 표시**: 합격/불합격 여부를 명확한 배지(Badge) 형태로 표시
- **반응형**: 모바일 환경에서도 테이블 가독성이 유지되도록 최적화
